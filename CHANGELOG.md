# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.0.0] - 2026-02-06

### Added
- Real-time task timing with live statistics
- Pause/resume functionality for breaks
- Task logging with automatic timestamps
- CSV export for detailed session data
- Session summary export
- Keyboard shortcuts (S, L, P, T, U)
- Light/dark theme toggle
- Automatic data persistence (localStorage)
- Responsive design for desktop and tablet
- Built-in help menu with documentation
- User name tracking with localStorage persistence

### Features
- Live table view of logged tasks with durations
- Real-time statistics: total rows, average time, session totals
- Undo functionality for accidental entries
- Timezone-aware timestamp recording
- Theme preference persistence
- No installation required—standalone HTML file

## [1.0.0] - Initial Release

### Added
- Basic task timing functionality
- Task logging
- Data export capabilities
- Initial UI implementation

## Upgrade Guide

### From 1.0.0 to 2.0.0

v2.0 introduces significant improvements and new features:

- **Performance**: Improved UI responsiveness
- **User Experience**: Added keyboard shortcuts and dark mode
- **Data Management**: Enhanced export options
- **Storage**: Better automatic data persistence

No breaking changes—existing data should work as expected.
